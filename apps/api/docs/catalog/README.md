# Product Catalog Foundation

Marketplace-independent product source of truth for CommerceAI OS.

## Capabilities

- Products, variants, SKUs, identifiers
- Hierarchical categories
- Flexible attributes
- Media metadata
- Search + pagination
- Lifecycle: draft → active → inactive → archived
