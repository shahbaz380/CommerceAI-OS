# Listing Management Foundation

Internal listing drafts, validation, templates, versioning, and marketplace mapping placeholders.

## Lifecycle

draft → ready_for_review → approved → scheduled → publishing → published  
(+ paused, ended, failed, archived)

**No live marketplace publish** in this module.
