# Variants

## Variant responsibilities

Variants represent SKU-level variations under a parent product record.

## Supported fields

- SKU
- title
- option values
- barcode
- MPN
- cost amount and currency
- weight and dimensions
- status
- metadata

## Rules

- Every variant owns its own SKU.
- Variant SKU must be unique within the workspace.
- Variant metadata and identifiers remain scoped to the parent product.
